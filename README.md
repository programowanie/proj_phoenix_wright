# Cencora Adrianna Joanna
## Symulacja rozprawy sądowej, 

Uczestniczą: 

- Oskarżony (alibi, dowody, przyznanie się do winy lub nie, zachowanie w sądzie, przeszłość kryminalna)
- Obrońca (z urzędu lub nie, doświadczenie, retoryka)
- Prokurator (doświadczenie, retoryka) 
- Sędzia (skorumpowany lub nie)
- Świadkowie (zeznanie na korzyść lub niekorzyść oskarżonego, wiarygodność)

Na początku program wykreuje postać oskarżonego - imię, nazwisko, wiek, a następnie przypisze do niego losowo jedno przestępstwo z puli (np.: kradzież w sklepie, profanacja kościoła, napad z bronią, gwałt, zabójstwo w afekcie lub nie, zabójstwo ze szczególnym okrucieństwem, pobicie, wypchnięcie ojca przez okno, handel narkotykami).

W dalszej kolejności prokuratorowi, obrońcy i Sędziemu losowo zostaną przyporządkowane cechy, mające wpływ na dalszy tok rozprawy, podobnie będzie ze świadkami.

Na podstawie powyższego Sędzia ogłosi wyrok (rok, 5 lat,.., dożywocie) lub uzna oskarżonego za niewinnego.
